# @firmanc/eslint-config-base

Adds ESLint settings and rules for JavaScript.

## Install

If you are using npm 5+

```bash
npx install-peerdeps --dev @firmanc/eslint-config-base
```

If you are using npm < 5

```bash
npm i --save-dev @firmanc/eslint-config-base eslint@8.5 eslint-plugin-import@2.25 eslint-plugin-prettier@4 prettier@2.5 eslint-config-prettier@8.3
```

## Usage

```JSON
// .eslintrc
{
  "extends": [
    "@firmanc/eslint-config-base"
  ]
}
```
